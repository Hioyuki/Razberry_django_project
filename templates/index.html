顔検出を実行

JPEG または PNG をアップロードすると、サーバー側で顔検出と人物照合を行います。 登録画像は known_faces/人物名/画像.jpg の構成を使います。

待機中: 画像を選んで顔検出を実行してください。
ここにアップロード画像のプレビューが表示されます。
ライブ映像を見ながら Raspberry Pi カメラで撮影し、そのまま解析できます。

セットアップメモ

まずは登録顔データを配置してください。登録画像が空だと照合はできません。

  1. 顔認証デモは http://127.0.0.1:8000/ または http://localhost:8000/ で開いてください
  2. サーバー起動コマンド: source .venv/bin/activate && python manage.py runserver 0.0.0.0:8000
  3. known_faces/Alice/alice_01.jpg のように人物ごとのフォルダを作る
  4. 無料の顔特徴分析モードはローカル実行なので追加課金はありません
  5. 判定したい画像をこの画面から選ぶ
  6. 顔検出を開始して結果を確認する
